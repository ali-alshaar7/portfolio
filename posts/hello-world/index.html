Code snippets will look like this:

def quantize(weights, bits=8):
    scale = weights.abs().max() / (2 ** (bits - 1) - 1)
    return (weights / scale).round().clamp(-(2**(bits-1)), 2**(bits-1)-1)

And inline code like memristor crossbar or torch.nn.Module should be easy to read.
